# reproductorAudioJava
Reproductor de archivos de audio programado en java. El el puedes añadir archivos de audio para reproducirlos o darle al boton obtener audios sistema para obtener la "musica" que hay en la unidad del directorio actual de donde se ejecuta este programa
Tambien puedes reproducir los archivos de forma aleatoria, ponerlos en bucle, ajustar el volumen, el progreso del archivo, y la velocidad: con un amplio selector de velocidades para los mas curiosos!
